# Task 4 - Sentiment Analysis

## Objective
Perform sentiment analysis on movie reviews using Natural Language Processing (NLP).

## Dataset
IMDb Movie Reviews Dataset (50,000 reviews)

## Technologies Used
- Python
- Google Colab
- Pandas
- Scikit-learn
- NLTK

## Steps Performed
1. Data Loading
2. Data Cleaning
3. Text Preprocessing
4. TF-IDF Feature Extraction
5. Train-Test Split
6. Logistic Regression Model Training
7. Model Evaluation
8. Sentiment Prediction

## Results
- High classification accuracy
- Successfully predicted positive and negative sentiments
